# TicTacToe Game

This is a tictactoe game built using python3.7. It's a terminal based application.

### Prerequistes:
* Python 3.7 required.
* It requires a colorama package.

### Installing package:
`pip install -r requirements.txt`

### To run the game use command:
`python3.7 tictactoe.py`
